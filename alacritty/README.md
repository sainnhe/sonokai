# Installation

Add this to `~/.config/alacritty/alacritty.yml`

colors:
  primary:
    background: "0x2d2a2e"
    foreground: "0xe3e1e4"

  normal:
    black: "0x1a181a"
    red: "0xf85e84"
    green: "0x9ecd6f"
    yellow: "0xe5c463"
    blue: "0x7accd7"
    magenta: "0xab9df2"
    cyan: "0xef9062"
    white: "0xe3e1e4"

  bright:
    black: "0x1a181a"
    red: "0xf85e84"
